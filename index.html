It Works!

This app will scrape out the fonts from google's font api and package them as a local bower component.

Usage:
choose your google font, once done, examine the link created:
<link href='http://fonts.googleapis.com/css?family=Open+Sans' rel='stylesheet' type='text/css'>
scrape out the bolded part of the url and append it here:

bower install --save https://google-fonts.herokuapp.com/googleFonts/The name you want?The font parameter

Example:

bower install --save https://google-fonts.herokuapp.com/googleFonts/openSans?family=Open+Sans

This will install the font into bower_components/The name you chose/the name you chose.css
