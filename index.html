This api acts as a Bower proxy to Google Fonts. It will scrape out the fonts from Google's api and package them as a local bower component.

Usage:

Choose your google font, once done, examine the link created:

<link href='http://fonts.googleapis.com/css?family=Open+Sans' rel='stylesheet' type='text/css'>

The scrape out the query string of the URL (bolded in the code snippet) and append it to the following: 

bower install --save https://google-fonts.herokuapp.com/googleFonts/nameOfPackage?queryString

Example:

bower install --save https://google-fonts.herokuapp.com/googleFonts/openSans?family=Open+Sans

This will install the font into: 

bower_components/The name you chose/the font you chose.css
